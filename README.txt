📋 รายละเอียด
   MyGameBot เป็นบอทอัตโนมัติสำหรับเกม Onmyoji (陰陽師)
   สามารถหารูปภาพบนหน้าจอและคลิกโดยอัตโนมัติ
   รองรับการทำงานแบบ Background (ไม่ยึดเมาส์)

   เมื่อเปิดโปรแกรม 
  Tab สำหรับเลือกโหมด
      - Tab "soul": โหมดปกติ (ใช้รูปภาพจาก assets/)
      - Tab "sougenbi": โหมดพิเศษ (ใช้รูปภาพจาก sougenbi/)
   
   c) สถานะการทำงาน
      ● สถานะ: ปิด (สีแดง)
      ● สถานะ: กำลังทำงาน (สีเขียว)
  
   d) ปุ่มควบคุม
      ▶ เริ่มทำงาน → เปิดบอทให้ทำงานอัตโนมัติ
      ■ หยุดทำงาน → หยุดบอทชั่วคราว

   หลังจากกดปุ่ม "เริ่มทำงาน" บอทจะ: 
   ข้อดี:
   ✓ ไม่ยึดเมาส์ - ใช้ Background Click API
